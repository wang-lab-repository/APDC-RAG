# APDC-RAG
APDC-RAG is a multimodal Retrieval-Augmented Generation (RAG) framework built for complex document Q&amp;A. It solves a common problem in multimodal AI: how to achieve high accuracy without wasting massive computing power on unnecessary image processing.
(The router model is open-sourced on ModelScope: [`J2001320/T5_router_final_model`](https://modelscope.cn/models/J2001320/T5_router_final_model))
